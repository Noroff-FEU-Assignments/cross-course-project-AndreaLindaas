Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

There are some ways to protect clients websites from intruders by applying web security best practices. They include:

  • Network security to protect the exchange of data across the network from breaches, intrusions and other threats.
  • sanitization of data which secures the data a system accepts by making sure it is in the right format. This can be done manually or by using a third-party solution.
  • secure authorization and authentication so it is hard to crack. To achieve this it is wise to use an open-standard authorization protocol or framework such as OneLogin etc.
  • Make sure to do research on any third party application you use.
  • Use Continuous Deployment/Continuous Integration (CI/CD) to automate processes like code tests, security checks and deployment of the application.
  • implementation of automatic user timeout to protect the users data.
